# 📊 Excel Analytics Platform

A full-stack web application that enables users to upload Excel files, analyze data, and generate interactive visual insights through dashboards.

---

## 📌 Features

* 📂 Upload and process Excel files (.xlsx, .csv)
* 📊 Data visualization using interactive charts
* 🔍 Data analysis with meaningful insights
* 📈 Dashboard for easy interpretation of data
* ⚡ Responsive and user-friendly interface

---

## 🛠️ Tech Stack

**Frontend:** React.js
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Other Tools:** Git, GitHub

---

## 💡 How It Works

1. User uploads an Excel/CSV file
2. Data is processed and stored in the database
3. Backend performs analysis on the dataset
4. Frontend displays:

   * Charts and graphs
   * Key insights
   * Analytical summaries
